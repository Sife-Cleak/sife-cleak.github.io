    torch.Tensor ¶

    译者：片刻小哥哥

    项目地址：https://pytorch.apachecn.org/2.0/docs/tensors

    原始地址：https://pytorch.org/docs/stable/tensors.html

    torch.Tensor 是包含单一数据类型元素的多维矩阵。

    torch.Tensor 是默认tensor类型 ( torch.FloatTensor ) 的别名。

    初始化和基本操作 ¶

    tensor可以从 Python list 或使用 torch.tensor() 构造函数：

    >>> torch.tensor([[1., -1.], [1., -1.]])
tensor([[ 1.0000, -1.0000],
 [ 1.0000, -1.0000]])
>>> torch.tensor(np.array([[1, 2, 3], [4, 5, 6]]))
tensor([[ 1, 2, 3],
 [ 4, 5, 6]])

    警告

    torch.tensor() 总是复制 data 。如果您有一个tensor“data”并且只想更改其“requires_grad”标志，请使用 requires_grad_() 或 detach() 以避免复制。如果你有一个 numpy 数组并且想要避免复制，请使用 torch.as_tensor() 。

    可以通过传递 torch.dtype 和/或 torch.device 来构造特定数据类型的tensor.device "torch.device") 到构造函数或tensor创建操作：

    >>> torch.zeros([2, 4], dtype=torch.int32)
tensor([[ 0, 0, 0, 0],
 [ 0, 0, 0, 0]], dtype=torch.int32)
>>> cuda0 = torch.device('cuda:0')
>>> torch.ones([2, 4], dtype=torch.float64, device=cuda0)
tensor([[ 1.0000, 1.0000, 1.0000, 1.0000],
 [ 1.0000, 1.0000, 1.0000, 1.0000]], dtype=torch.float64, device='cuda:0')

    有关构建tensor的更多信息，请参阅Creation Ops

    可以使用 Python 的索引和切片表示法来访问和修改tensor的内容：

    >>> x = torch.tensor([[1, 2, 3], [4, 5, 6]])
>>> print(x[1][2])
tensor(6)
>>> x[0][1] = 8
>>> print(x)
tensor([[ 1, 8, 3],
 [ 4, 5, 6]])

    使用 torch.Tensor.item() 从包含单个值的tensor中获取 Python 数字：

    >>> x = torch.tensor([[1]])
>>> x
tensor([[ 1]])
>>> x.item()
1
>>> x = torch.tensor(2.5)
>>> x
tensor(2.5000)
>>> x.item()
2.5

    有关索引的更多信息，请参阅索引、切片、连接、变异操作

    可以使用 requires_grad=True 创建tensor，以便 torch.autograd 记录对其的操作以进行自动微分。

    >>> x = torch.tensor([[1., -1.], [1., 1.]], requires_grad=True)
>>> out = x.pow(2).sum()
>>> out.backward()
>>> x.grad
tensor([[ 2.0000, -2.0000],
 [ 2.0000, 2.0000]])

    每个tensor都有一个关联的 torch.Storage ，用于保存其数据。tensor类还提供存储的多维跨步视图并定义数值对其进行操作。

    笔记

    有关tensor视图的更多信息，请参阅 tensor视图 。

    笔记

    有关 torch.dtype 、 torch.device 和的更多信息torch.layout torch.Tensor 的属性，请参阅 Tensor Attributes。

    笔记

    改变tensor的方法用下划线后缀标记。例如，“torch.FloatTensor.abs_()”就地计算绝对值并返回修改后的tensor，而“torch.FloatTensor.abs()”则计算产生一个新的tensor。

    笔记

    更改现有tensor的 torch.device 和/或 torch.dtype ，考虑在tensor上使用 to() 方法。

    警告

    torch.Tensor 的当前实现引入了内存开销，因此可能会导致具有许多微小tensor的应用程序中内存使用量异常高。如果这是您的情况，请考虑使用一个大型结构。
