    torch.nn.init ¶

    译者：片刻小哥哥

    项目地址：https://pytorch.apachecn.org/2.0/docs/nn.init

    原始地址：https://pytorch.org/docs/stable/nn.init.html

    警告

    该模块中的所有函数都旨在用于初始化神经网络参数，因此它们都在 torch.no_grad() 中运行模式，并且 autograd 不会考虑。

    警告

    为了实现自归一化神经网络，你应该使用 nonlinearity='linear' 而不是 nonlinearity= 'selu' 。这为初始权重提供了1 /N 的方差，这是在前向传播中引入稳定固定点所必需的。相反，SELU 的默认增益牺牲了归一化效果以获得更稳定的梯度呈矩形层状流动。
