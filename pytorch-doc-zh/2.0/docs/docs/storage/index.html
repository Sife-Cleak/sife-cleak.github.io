    torch.Storage ¶

    译者：片刻小哥哥

    项目地址：https://pytorch.apachecn.org/2.0/docs/storage

    原始地址：https://pytorch.org/docs/stable/storage.html

    torch.Storage 是与默认数据类型对应的存储类的别名 ( torch.get_default_dtype() )。例如，如果默认数据类型为 torch.float ，则 torch.Storage 解析为 torch.FloatStorage 。

    torch.<type>Storage 和 torch.cuda.<type>Storage 类，如 torch.FloatStorage 、 torch.IntStorage 等实际上并未实例化。调用它们的构造函数会创建一个具有适当的 torch.dtype 和 torch.dtype.device](tensor_attributes.html#torch.device "torch.device").torch.Storage类具有与 [torch.TypedStorage`](#torch.TypedStorage "torch.TypedStorage") 相同的所有类方法。

    torch.TypedStorage 是特定 torch.dtype ，并且内部数据将被适当地解释。 torch.TypedStorage 包含一个 torch.UntypedStorage，它将数据保存为无类型字节数组。

    每个跨步的 torch.Tensor 都包含一个 torch.TypedStorage ，它存储所有数据torch.Tensor 视图。

    警告

    将来，除了 torch.UntypedStorage 之外的所有存储类都将被删除，并且 torch.UntypedStorage 将被删除。在所有情况下都使用。

    如果共享为 True ，则内存在所有进程之间共享。所有更改都会写入文件。如果共享为 False ，则存储上的更改不会影响文件。

    size 是存储中元素的数量。如果共享为 False ，则文件必须至少包含 size * sizeof(Type) 字节( Type 是存储类型)。如果共享为 True，则将在需要时创建文件。

    参数 * filename ( str ) – 要映射的文件名 * shared ( bool ) – 是否共享内存 * ** size** ( int ) – 存储中的元素数量
